# Dark-Tools

简体中文 | [English](./README_EN.md)

Dark-Tools 是一个功能强大的工具库，旨在简化和加速开发过程。该库包含一系列实用的工具和功能，帮助开发人员更高效地编写代码、调试和维护项目。

## 特性

- **日志记录**：封装了 `spdlog` 的日志库，支持格式化输出，方便记录和调试。
- **易于集成**：提供简单易用的接口，方便集成到现有项目中。
- **高性能**：优化的实现，确保在各种应用场景下都能提供高性能。
